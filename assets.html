# Static Resource

The management of static resources is not the responsibility of Vapper, Vapper is only responsible for SSR, essentially, because Vapper uses Vue-CLI or Poi to manage webpack configuration. So all the way resources are handled follows Vue-CLI and Poi:

  • Vue CLI - Static Assets Handling
  • Poi - Using the Public Folder
